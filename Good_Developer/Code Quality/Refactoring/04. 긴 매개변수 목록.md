# 4. 긴 매개변수 목록
- 매개변수가 많아지면 함수간에 구분하기가 어려워진다.

</br>

### 매개변수를 질의 함수로 바꾸기
- 어떠한 매개변수를 다른 매개변수를 통해 알 수 있는 경우 사용한다.(중복 매개변수)

</br>

### 플래그 인수 제거하기
- 플래그는 보통 조건문에 사용한다.
- 플래그를 사용한 함수는 차이를 파악하기 어렵다.(메서드에 직접 들어가 봐야 안다.)
- 조건문 분해하기를 활용할 수 있다.

</br>

### 여러 함수를 클래스로 묶기
- 비슷한 매개변수 목록을 여러 함수에서 사용하고 있다면, 해당 메소드를 모아서 클래스를 만들 수 있다.
